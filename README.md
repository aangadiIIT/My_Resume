A premium, high-performance resume website built with **Node.js, Express, and EJS**, featuring a modular data architecture and a stunning **Glassmorphism UI**.

## 🚀 Key Features
- **Dynamic Data Hub**: Modular JSON architecture for easy updates.
- **Glassmorphism UI**: Premium design with blurred backgrounds and smooth animations.
- **Interactive Skills Radar**: Visual expertise distribution using Chart.js.
- **Headless Contact Form**: Custom-styled form relaying directly to Google Forms.
- **Dark Mode Persistence**: Automatic theme detection and `localStorage` persistence.
- **SEO Optimized**: Complete JSON-LD structured data for better search visibility.

## 🛠️ Technical Stack
- **Backend**: Node.js, Express.js
- **Frontend**: EJS, Bootstrap 5, Vanilla CSS (Glassmorphism)
- **Visualization**: Chart.js
- **CI/CD**: GitHub Actions
- **Security**: Helmet.js for hardened HTTP headers

## 📦 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/akhilangadi/My_Resume.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm start
   ```

## 🚢 Deployment
This project is configured for automated deployment via **GitHub Actions**.
- **Current Target**: Azure Web App (`akhilesh-resume`)
- **Workflow**: `.github/workflows/main_akhilesh-resume.yml`
